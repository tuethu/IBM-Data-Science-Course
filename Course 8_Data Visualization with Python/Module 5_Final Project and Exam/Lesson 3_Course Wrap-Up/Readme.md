- Which best practice should you follow to ensure effective data visualization?<br/>
Label the axes clearly, provide context, and use only necessary data and labels.  

- What are the layers of the Matplotlib architecture?<br/>
Backend Layer, Artist Layer, and Scripting Layer. 

- What does a histogram illustrate?<br/>
Data distribution within specific intervals, allowing us to understand its shape and concentration.

- What is the code for the Matplotlib magic function?<br/>
%matplotlib. If the plot gets generated in a new pop window, you can enforce generating plots into a browser using this code.

- For what purpose is an area plot commonly used?<br/>
An area plot displays the magnitude and proportion of multiple variables and compares two or more variables.

- What is a way of statistically representing the distribution of the data through five main dimensions?<br/>
A box plot is a way of statistically representing given data distribution through five primary dimensions. These include Minimum, First quartile, Median, Third quartile, and Maximum.

- Fill in the blank. A _________displays two variables against each other.<br/>
Scatter plot 

- Where can Waffle Charts be applied?<br/>
The  allocation of budgetary resources across categories or departments within an organization.

- What is Seaborn?<br/>
Seaborn is a data visualization library based on Matplotlib. Its primary purpose is to build a high-level interface for drawing attractive statistical graphics, including regression plots, box plots, and more.

- Folium is a powerful data visualization library in Python that primarily helps people visualize geospatial data.<br/>
What is its default map style? Open Street Map

- What feature enhances interactivity and provides contextual information to maps?<br/>
Markers represent specific locations or points of interest, providing additional information when clicked. Markers are like signposts that guide us through the map, highlighting essential elements.
